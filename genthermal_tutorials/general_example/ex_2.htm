Exercise 2 - Disc Stress Analysis

This problem is a continuation of the first exercise - to determine the effect of the disc geometry change on the component stresses.

Exercise Objectives

To examine the effect of a change in disc cob width on the stress distributions of the HPT disc.

Problem

Given the basic model described in the preceding sections (example.pm), and the reduced disc cob width model of exercise 1 (exercise1.pm), determine:

  • The time in the square cycle at which the disc maximum hoop stress occurs

  • The change in disc maximum hoop stress

  • The change in worst principal stress

 

Instructions

The stress analysis can be completed using either of the following methods:

  • Running a combined transient thermo-mechanical analysis

  • Using temperature distributions interpolated from the results of Exercise 1

If time allows, complete the analysis using both techniques and discuss their relative merits with your training supervisor.

Use the model from exercise 1 and complete a combined transient thermo-mechanical analysis.

It is recommended that structural analysis be disabled for all components other than the HPT disc/blade assembly.

Use the model from exercise 1 to create a new model of the HPT disc/blade assembly only, and complete a series of single time point stress analyses using temperature distributions interpolated from the results of Exercise 1.

  • Pressure loads will need to be applied if the thermal boundary conditions are not present.

Do a time plot of a point at the centre of the bore of the disc, determine the maximum hoop stress, and plot the contours at this point.

The following contour plots should be generated for the HPT disc in both cases:

  • Plot 1:  Hoop stress at maximum hoop stress condition - original cob

  • Plot 2:  Worst principal stress in the XY plane at maximum rim to cob temperature difference - original cob

  • Plot 3:  Hoop stress at maximum hoop stress condition - reduced cob

  • Plot 4:  Worst principal stress in the XY plane at maximum rim to cob temperature difference - reduced cob
