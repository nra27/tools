To determine how to change the boundary conditions

Note that in this example matching exercise, only the factor of the heat transfer correlations needs to be changed.

Print off difference plots defined in graph layout

Print off a plot of the HP turbine disc with the boundary conditions

For each boundary condition, mark on the metal and air temperatures at the maximum take off point (MTO) (Ramp 5, step 1). These can be obtained from the summary file, or from discrete data display

From the plots, determine whether the factor for the heat transfer coefficient should be increased or decreased. Note that for most of the factors, accuracy to one decimal place is used.

Make the change (initially make large changes to the factor, e.g. 2 or 0.5 to determine which factor makes what kind of a difference) and run the model again.  

Repeat the process, making a note of which changes had which effect until the perfect match is achieved
