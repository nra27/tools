Introduction to the General Example Problems

 

The general example problems are part of the SC03 Thermal Analysis Training.

This example covers typical problems that are likely to be encountered in actual gas turbine engine project work.

The objective of the examples is to build on the basic skills learnt from the SC03 introductory tutorials and develop understanding of the results that are generated from an SC03 analysis.

In each case a copy of the analysis model is provided, and the engineer is required to make changes to assess the effects on component temperature distributions, stresses, tip clearance etc. This means opportunity is given to develop skills and experiences without the time consuming process of creating a completely new model, and eliminating risks associated with training new personnel on main engine projects.

Descriptions of the thermal and thermo-mechanical models used in the exercises should now be reviewed before starting one or more of the problems in line with training needs.

Note that some uses of the mouse can cause the SC03 session to 'lock up'. (e.g. selecting a drop-down menu item from a right-hand mouse click which is outside the main SC03 window.) The only solution is to log onto another machine and kill the session. This will cause all unsaved changes to be lost. To avoid unnecessary loss, it is important to save your model often. (e.g. use two different file names and save alternately to them.) Click here to go the the Best Practice to explain how to kill an SC03 session from another machine. It is suggested that this Best Practice is printed out.
