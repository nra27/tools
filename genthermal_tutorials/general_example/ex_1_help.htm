Locations of reference points

 

Live and Dead Rims

To decide where to position the reference points for a temperature response from parts of the turbine blade, a little must be understood about live and dead rims in a turbine blade. 

The live rim is the last piece of hoop-carrying (continuous) metal at the edge of the disc, going radially outwards, just where the fir tree root begins.

The dead rim is at the top of the fir tree fixing, where there is no circumferential piece of metal.

Reference points to give the temperature at the top of the disc should be on the last live rim, as this is the last piece of continuous hoop-carrying metal.

Creating plots from two different models on a single graph

Key Value Approach

The recommended way is to use Key Values.

  1. In the original model, set up key values to define the transient temperature output that is required 
    e.g. cob1=tk@cob1
    Note that you must already have defined reference points (from the Model menu) in the desired position, e.g. cob1

  2. Run the original model (example.pm) thermally

  3. Save an item file, and select the graphical items that have been produced automatically from the key value definitions.
    e.g. key value cob1 will produce the graphical item grcob1

  4. Run the reduced cob model to provide temperatures

  5. Read in the .item file stored in (3) using the item tab from Open file.

  6. Define a reference point on cob, e.g. cob2

  7. A simple plot of both temperatures can be produced as follows using Define Graph Layout. The method for producing this graph is shown in detail in the 2D SC03 tutorial within the capability Intranet.

 

Graph G2
Subgraph S2
Plot Variable1 GRcob1(t)
Plot Variable 2 tk@cob2

 

Export Subgraph Approach

  1. Run the original model (example.pm) thermally

  2. Define a reference point (from the Model menu) in the desired position, e.g. cob1

  3. Use the Define graph layouts option to produce a graph of temperature versus time. The method for producing this graph is shown in detail in the 2D SC03 tutorial within the capability Intranet.

 

Graph G1
Subgraph S1
Plot Variable1 tk@cob1

 

  1. When the graph has been plotted, from the File menu on the graph window select Export Subgraph
    A tabular output will be produced with variables time (t) and temperature GRcob1 (i.e. puts a GR prefix in front of plot variable cob1)

  2. Save the file as a .item file.

  3. Run the reduced cob model to provide temperatures

  4. Read in the .item file stored in (5) using the item tab from Open file.

  5. Define a reference point on cob, e.g. cob2

  6. A simple plot of both temperatures can be produced as follows using Define Graph Layout as before:

 

 

Graph G2
Subgraph S2
Plot Variable1 GRcob1(t)
Plot Variable 2 tk@cob2
