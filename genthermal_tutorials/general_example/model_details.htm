Thermal/Thermo-Mech. Model Details

The thermal model and associated files used for these exercises should be copied to your local file system. 

If the exercise files have not already been copied, you should do so now using the following command:

cp {path/filename} {new or existing filename}

The file required is the gneral example database:

/usr/rreng/data/sc77/general.example.start.pm

If you cannot access this directory, the required files are stored on the web.

 

Geometry

The model provided has been devised to represent a simple gas turbine spool.

The relevant procedure can be referred to here.

A single stage drilled rim HP turbine, linked to a 3 stage axial compressor drum is assumed, with an internal cooling air system similar to the RB211-535E4 engine as shown below.

Performance Data and Cycle Definition

The engine performance data standard determines the engine annulus conditions and, indirectly, the internal cooling air system requirements. Thus, the boundary conditions for the thermal model are ultimately derived from the performance data.

For the examples, the RB211-535E4 basic design document TDR58772 has been used. A simple square cycle is to be considered. The cycle definition is contained within the SC03 analysis model. Refer to the Analysis/Cycle Definition option in SC03 to see full details.

Note that VW34 would usually be used to generate a .bdd file for input of the cycle definition and performance data to SC03. This method is in the 2D sc03 tutorial. A copy of the performance data is also provided with the example files.

Boundary Conditions & Boundary Definitions

Cooling air system data is provided at the design point (ISA SLS Max. Take-Off). Expressions have been derived and assigned to the model geometry via the boundary streams, ducts, voids etc.

All expressions can be reviewed via SC03 using the command File/Save/Summary.

A copy of the Summary Print File for the thermal model is also provided as one of the example files.

The thermal boundary definitions should also be reviewed through an SC03 session.

Material Specification

Materials for each component have been specified as shown in the model:

Component Material Code
HPC Discs Titanium TDX
HPC Blades Inco 901 QDU
HPT Disc Waspaloy QDY
HPT Blades Mar-M-002 QPD
Combustor Casings Jethete M152 EBP
Casings C1023 QAB
Shaft CMV AEF

 

Properties for these materials can be examined through the Summary Print or interactively in SC03.
