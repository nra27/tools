Aims

The overall aim of this training module is to enable new users of SC03 to acquire the knowledge and skills necessary to create 2D thermal analysis models.

Module Objectives

When you have worked through all of the sections contained in the module you will be able to identify and describe the main stages in the construction of 2D thermal models within the SC03 environment.

Specific sections of the module will enable you to:

  • explain how program options and commands are accessed in SC03

  • explain what is meant by "chaining" geometry

  • list the steps undertaken in the automatic "meshing" process

  • explain how cycle definition and performance data from a .bdd file can be retrieved and modified in SC03

  • define boundary conditions in SC03

  • state the functions performed by the Thermal Analysis controls in SC03

  • describe the methods used to complete a thermal analysis and state what action should be taken if an analysis has failed

  • describe the ways in which results can be processed and represented in SC03

1. Using the training module

SC03 2D Thermal Modelling forms part of the SC03 Thermal Analysis Training System which provides the inexperienced analyst with step-by-step interactive training in the use of 2D and 3D modelling techniques.

The training modules are designed to be used in conjunction with the SC03 program and therefore they cannot be used by PC users (unless on a Technical PC). 

As you work through the training material, you should have an SC03 session running.  In this way, you will be able to get step-by-step instructions from the training material and to carry out these instructions in the SC03 environment.

2. Navigating from one part of the training module to another

You will navigate through the module by

  • clicking on hyperlinks, or

  • choosing options from the main menu screen

Hyperlinks can be easily recognized, as they consist of blue text (which is underlined when the mouse hovers over it) and are used extensively to move around the World Wide Web.  When you click on a hyperlink you are taken to the information indicated in the text of the hyperlink.
