Creating More Lines

The line will now have been drawn by the sc03 program. To see this line follow these commands:

Close the Line command by holding down the right mouse button and selecting End in the sub-menu.

 

 

Open the display menu by selecting Display with the left mouse button, then choose the Display Settings option

 

 

Ensure that the following settings are selected on the Display Settings panel:   C-Contours, F-Geometry Faces, G-Geometry, O-Mesh Outlines, P-Physical Geometry, R-Errors, W-Whole Model, X-Axis, Y-Properties and the 2D Mode.

SC03 will automatically position the line on the screen and will adjust the position of the model as it is constructed. The line can be manually adjusted using the Page Up and Page Down keys on the keyboard. It can also be moved using the centre mouse button.

If you make any mistakes when constructing the geometry, then carry out the following steps:

  1. The part of the geometry (or entity) that you want to delete should be highlighted either by clicking the left mouse button on the entity, or by creating a drag-box around the geometry to be deleted by holding down the left mouse button and dragging between the diagonal corners of the required drag box.  The entities to be deleted will be highlighted in yellow.

  2. Move the arrow into the 2D Geometry Commands list, and choose Delete 2D Geom with the left mouse button.

  3. The highlighted geometry will be deleted.

Using the same method of co-ordinate input shown previously, complete the geometry for the disc using the co-ordinates shown below.

Close the Line command by holding down the right mouse button and selecting End in the sub-menu.

 

Creating Rectangles

Earlier in this section you created the geometry for the disc, using the Draw Line option.
We will now create the geometry for the shaft, using the Create Rectangle option.

Select the Rectangle option in the 2D Geometry Commands panel, with the left mouse button. A cross should now appear.

Hold the right mouse button and select Co-ords from the sub-menu, and type in the BOTTOM LEFT co-ordinates from the table on the right, and select OK.

Now select Co-ords option again and type in the TOP RIGHT co-ordinates to form the rectangle, then select OK.

Bottom Left (x,y)

Top Right (x,y)

(0,100)

(230,120)

Close the Rectangle command by holding down the right mouse button and selecting End in the sub-menu.

The rectangle has now been formed.

After inputting the co-ordinates select the Settings option from the 2D Geometry Commands panel and press the Reset button

 

Chain geometry

At the moment, the geometry is just a series of lines. The process of joining them together is called chaining.

Choose the Chain 2D Geom option in the 2D Geometry Commands box.

SC03 will attempt to chain the geometry, and one of the following three boxes will now appear:

    • If one of the first two boxes appear, the following error message will appear on the SC03 screen.

If unconnected entities are found (Box 1), do not click Yes to delete the entities, unless you are absolutely sure that the highlighted geometry should be deleted.  It is often the case that this error message appears where two entities are unconnected where they should be connected - therefore the lines should not be deleted.  Click No.

Click OK on the warning box (Box 2 above) and OK on the information box (Box 3 above).

You will now need to make appropriate corrections, depending on  what error message is showing on the geometry.

    • In this training course the top geometry line was purposely missed off so that the chain option would fail.

We will now draw this line in, using the Draw Line command.

Select the 'Line' command out of the 2D Geometry Commands box.

Move to the top of the geometry where the line is missing, and press the left mouse button close to the geometry, The draw command will now start a yellow line at the edge of the geometry

Move the end of the line to the finish position and press the left mouse button.

SC03 will now have drawn the line specified, and the End option in the sub-menu (right mouse button) can be chosen.

Now choose the Chain 2D Geom option in the 2D Geometry Commands box.

Mark Internal Lines

If you refer back to the problem described in the User Guide, you will notice that the geometry has cooling holes shown.

 

On 2D geometry the holes would be shown as internal lines, which must be specified before any running/meshing takes place. (Internal lines are lines between components; external lines separate a component from the fluid.)

Marking internal lines can be done either automatically or manually by detect. For simple shapes, automatic marking is used and is usually reliable.   Manual marking is used for more complex forms.

The internal lines can be marked as follows:

Select the Mark Int 2D Geom symbol inside the 2D Geometry Commands box with the left mouse button.

A box will now appear similar to this one:

 

Select Manual for this geometry, and click on OK using the left mouse button.

A cross will now appear.

Move the cross using the mouse and detect with the left mouse button the two internal lines shown on the right.

Then hold the right mouse button and choose END on the sub-menu.
 

  • An error box may be displayed if the geometry is incorrect.  SC03 will show you where the errors are on your geometry, and corrections can then take place.

 

If the geometry has no errors, the internal lines chosen will change colour, and the geometry is then ready for meshing.

It is a good idea to save your database now before continuing with the training module. 

Select File/Save and then select the Database tab.  Type in a filename ending in .pm

 

Automatic Meshing

The automatic meshing in SC03 generates either:

  • 3D 10 noded (quadratic) tetrahedral elements or:

  • 2D 6 noded (quadratic) triangular elements.

To generate a 2D mesh, the major steps completed automatically by SC03 are:-
1. Geometry Recognition
2. Node Generation
3. Surface Triangulation

Go to the SC03 Users Guide for further details of the automatic meshing control.

We will now generate a mesh on the geometry we have just constructed in SC03.

Open the Mesh Menu by selecting Mesh with the left mouse button

 

Select the Generate Mesh command

 

 

 

 

 

 

 

The Meshing Control Panel will then be displayed

 

Make sure that the default mesh settings are used as shown on the right then select OK with the left mouse button.

Edge Node Spacing

1.0E+06
Internal Nodes OFF
Distortion Ratio 5
No. Largest Faces 0
Elements Tri 6
Diagnostics OFF

The 2D geometry will then be automatically meshed, and as the mesh generation progresses, the main meshing steps can be observed.

 

Every SC03 command panel includes a help button. Click on the help button at any time to see the Users Guide help section for that command.

 

Use your model viewing skills to better visualise the generated mesh.
