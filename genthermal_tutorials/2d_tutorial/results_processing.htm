SC03 2D Thermal Modelling

Section 7: Results Processing

Objectives of this Section

When you have completed this section you will be able to:

  • list the types of data that must be checked for accuracy

  • describe 4 different ways in which results can be processed and represented in SC03

Learning Points of this Section

  1. Checking the model results

  2. Displaying and presenting the results by means of:

    • Contour plots

    • Result values (Get Values)

    • Time plots

    • User-defined graphs

     

Check Results

It is very easy to believe component temperatures predicted by the analysis program, but they are totally dependent on the accuracy of the boundary condition assumptions.

An extensive check of all of the following data must be completed, and where possible confirmed by hand calculation, or by comparison with air/oil system model data.

Data for Checking

  • fluid type (boundary conditions other than air only)

  • mass flows

  • fluid temperatures

  • fluid pressures

  • heat transfer coefficients

  • material type

There are a number of facilities available to aid the checking process. These are explained below.

Summary Print

A thorough check of all evaluated thermal boundary condition data can be completed by generating a summary print (.prt) file of the analysis results.

Open the File Menu by selecting File with the left mouse button then select the Save File command with the left mouse button and click the Summary tab on the Save File panel.

Enter a summary file name in the panel (ending with .prt) and click OK.

In the Print File Contents panel select the All option for the Thermal Results.

In the Thermal Results Units panel select pure MKS (or your preferred units system).

 

Available unit systems are shown below:

 

Name Flow Speed Pressure Temp. htc Flux
SC03 Mg/s RPM MPa K mW/mm2/K mW/mm2
MKS kg/s RPM kPa K W/m2/K W/m2
Mixed lb/s RPM lbf/in2 K W/m2/K W/m2
Alt Mixed kg/s Rad/s kPa F thu/tf2/F Bthu/ft2

A listing of all calculated boundary condition values will be written to the .prt file, and a window opened displaying the contents of the file.

The summary print file contents include results at every time point.

Check all thermal boundary condition values are as expected at a key time-point e.g. steady state max. conditions.

 

Process Thermal Model Results 

Once an analysis model has been fully checked, the results may be processed.

There are a number of facilities available via the Results, Display and Graph Menu commands to review the results:

  • Contour Plots

  • Get Values

  • Time Plots

  • User Defined Graphs

We will now generate each of these plots in turn.

Contour Plots

To generate a contour plot of the analysis model results:

 

Open the Results menu by selecting Results with the left mouse button

 

 

 

then select the Contour Types command and the Temperature (Temp) panel. Select the TK Temperature (K) option then click OK with the left mouse button.

 

The calculated temperature distribution will then be displayed.

The metal temperatures at the current time point/load case (t=3000 secs) will be shown.

 

To change the cycle time at which contours are displayed:

Open the Results menu by selecting Results with the left mouse button, then select the Load Case Control command and enter the appropriate Ramp and Step numbers.

To change the style of contours that are used:

Select the Contour Styles command and enter the appropriate style.

To change the contour colours:

Select the Contour Colours command and enter the colours required.

 

Get Result Values

The contour plots provide a qualitative understanding of the predicted temperature distribution of a component. However, contour plots should not be used to determine the absolute value at a point on the model.

To determine the temperature (or any results type) at a specific point on the model:

Open the Results Menu by selecting Results with the left mouse button then select the Contour Types command. Select Temp and then Temperatures TK - temperatures in Kelvin

Open the Results Menu by selecting Results with the left mouse button then select the Get Values command.

The cursor will change to an arrow, which - when clicked over the contoured results - will give the values at the nearest mesh node.

Select a point on the model and note how the result is tabulated on the screen.

Values can continue to be selected until End is selected with the right mouse button.

To change the cycle time at which values are displayed:

Open the Results Menu by selecting Results with the left mouse button, then select the Load Case Control command and enter the appropriate Ramp and Step numbers.
