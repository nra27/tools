SC03 2D Thermal Modelling

Section 2: 2D Geometry Construction & Meshing

Objectives of this Section

When you have completed this section you will be able to:

  • give examples of settings (in the 2D geometry palette) which need to be adjusted for 2D geometry manipulation

  • identify the main stages involved in creating 2D geometry

  • explain what is meant by "chaining" geometry

  • list the steps undertaken during the automatic meshing process

Learning Points of this Section  

  1. Accessing the 2D Geometry commands list and adjusting settings

  2. Using coordinates to draw lines

  3. Using the Create Rectangle option to create geometry

  4. Chaining geometry

  5. Marking internal lines

  6. Automatic meshing

 

Basic 2D Geometry Construction

Before beginning this section you should already have an SC03 session open. If not, return to the section on how to start an SC03 session. Click here if SC03 is not yet running

In this section we are to construct the 2D geometry for the problem described in the User Guide. To do this we use 2D geometry commands.

Open the Model Menu by selecting Model with the left mouse button

then select 2D Geometry Commands with the left mouse button. 

 

This panel can be 'torn off' to float over the window in a similar way to other 'tear-off' menus.

A floating palette of names will now appear on the sc03 screen similar to the one shown below. This box can be moved around the screen, using the mouse.

The 2D geometry palette includes a settings option, which enables control of options relevant to 2D geometry manipulation.

Tolerance is normally set automatically as 1/3 of the length of the shortest entry. If the AXISYMMETRIC flag is set on, geometry found with a negative Y co-ordinate will cause an error.

 

Click CANCEL to close the settings box.

 

Creating Lines

To start constructing the geometry, move the arrow into the 2D Geometry Commands box, and choose Line with the left mouse button. A cross will now appear on the SC03 screen. We will draw our geometry lines by using co-ordinates, as this is more precise than the other options.

Press & Hold the right mouse button then select the Coords option in the sub-menu.

 

Type in the STARTING co-ordinates

 

Select OK using the left mouse button.

You have now specified the starting co-ordinates of the first geometry line. A yellow line should now have appeared (which can be moved around the screen). This line needs a finishing point, so:

Press the right mouse button and choose Coords option again. Type in the FINISHING co-ordinates.

Select OK again using the left mouse button.
