Retrieval of SPAN data into VW34

This part of the training module details how to retrieve and/or scale data from the Cooling Air Network system SPAN for inclusion into VW34 and the BDD file.

Select the SPAN Data option in the VW34 Program Control Panel

 

 

 

 

 

The Span Directory panel will now appear

 

 

 

 

 

 

 

Type /usr/rreng/data/sc77/ in the selection box of the Span Directory panel or navigate through the directories using the mouse, then click on OK

If you cannot access this directory, the required files are stored on the web.

The Read Output panel will now appear

 

 

 

 

 

 

Fill in the Read Output panel as shown here and Click on OK.

Engine No. 8357

Verse No. 1
Chapter BDD Data Units (Option) Imperial

The main VW34 display window should now return the following messages:

Span Design Point Scaling - Air System 1
Reading Span database - Please wait ...
SPAN database successfully read

Reading BDD design point - Please wait ...
BDD design point successfully read

WARNING: Bit H200 has no data associated with it
WARNING: Bit H220 has no data associated with it
WARNING: Bit V103 has no data associated with it
WARNING: Bit V130 has no data associated with it
WARNING: Bit V200 has no data associated with it
WARNING: Bit VI14 has no data associated with it

 

  • Ignore these warning messages as they are not of any relevance to us 

    • The following two panels will now appear: the Span Input Panel 1 (for Span Node Temperatures) and the SPAN Output Window which displays the node temperature data retrieved from the SPAN database.

    Click on the OK button as we do not need node temperatures for this exercise.

    The main VW34 display window should now return the following messages:

    Calculating node temperatures at time pt 0
    Calculating node temperatures at time pt 60
    Calculating node temperatures at time pt 1000
    Calculating node temperatures at time pt 1010
    Calculating node temperatures at time pt 2000
    Calculating node temperatures at time pt 2010
    Calculating node temperatures at time pt 3000

    Another two panels will now appear: the Span Input Panel 2 (for Span Node Pressures) and the SPAN Output Window which this time displays the node pressure data retrieved from the SPAN database.

    In SPAN Input Panel 2 enter the following information:-

    Default Pup: p30 and Default Pdn : p26

    You will now need to copy the information about Nodes C01 to C06 from the SPAN Output Window to the SPAN Input Panel 2.

    To copy the information:

    use the left mouse button to highlight Node C01 in SPAN Output Window, point to the appropriate box under the Span Node heading in SPAN Input Panel 2, and finally click the centre mouse button to copy the information.

    The values for Pupstream, Pdownstream and Scale Method  will be copied into SPAN Input Panel 2 when you move to the next row of text boxes with the mouse.

    Repeat for Nodes C02 to C06 (you will have to use the More button to complete the panel), which should leave you with the following values in the Panel:-

    Default Pup: p30 and Default Pdn : p26

    Span Node Pupstream Pdownstream Scale Method
    C01 p30 p26 1
    C02 p30 p26 1
    C03 p26 p24 1
    C04 p30 p26 1
    C05 p30 p26 1
    C06 p30 p26 1

     

  • The Pupstream and Pdownstream values for Node C03 have been manually edited (using the keyboard) after copying from SPAN Output Window.

    • Click on the OK button to display the next screens

    The main VW34 display window should now return the following messages:

    Calculating node pressures at time pt 0
    Calculating node pressures at time pt 60
    Calculating node pressures at time pt 1000
    Calculating node pressures at time pt 1010
    Calculating node pressures at time pt 2000
    Calculating node pressures at time pt 2010
    Calculating node pressures at time pt 3000

    The following two panels will now appear: the Span Input Panel 3 (for Span Link Mass Flows) and the SPAN Output Window which displays the link mass flow data retrieved from the SPAN database.

     

    In SPAN Input Panel 3 enter the information shown on the right.

    Default Ref. Flow:

    w26

    Default Pup :

    p26

    Default Tup:

    t26

    Default Pdn:

    p44

    You will now need to copy the information about Links L001 and L002 from the SPAN Output Window to the SPAN Input Panel 3.

    To copy the information:

    Use the left mouse button to highlight Link L001 in SPAN Output Window

    Point to the appropriate box under the Span Link heading in SPAN Input Panel 3

    Finally click the centre mouse button to copy the information.

    The values for Ref. Flow, Pupstream, Tupstream, Pdownstream and Scale Method will be copied into SPAN Input Panel 3 when you move to the next row of text boxes with the mouse.

    Repeat for Link L002.

     

    This should leave you with these  values in the Panel:-

    Default Ref. Flow:

    w26

    Default Pup :

    p26

    Default Tup:

    t26

    Default Pdn:

    p44

    The main VW34 display window should now return the following messages:

    Calculating link mass flows at time pt 0
    Calculating link mass flows at time pt 60
    Calculating link mass flows at time pt 1000
    Calculating link mass flows at time pt 1010
    Calculating link mass flows at time pt 2000
    Calculating link mass flows at time pt 2010

    A panel will now appear that prompts for a second SPAN air system

     

  • Click on Cancel to finish the generation of Span data
      

  • *** Do not click on the OK button as this will cause VW34 to crash***

    • The main VW34 display window should finally return the following message:

    Generation of Span data complete
    Writing a .bdd file

    The cycle definition and performance data generated within VW34 are passed to SC03 using a .bdd file.

    Select the Write BDD Data option from the Program Control panel.

    A Write File panel will  then appear.

    Define the directory and file names, then select OK to write the .bdd file.

     

     

     

     

     

     

     

     

    Ending VW34

    Note that VW34 also includes the ability to define:

    • Graphical Data and

    • Fluid Properties

    To end your VW34 session:

     

    Select the QUIT button on the main VW34 message window and confirm that VW34 should terminate.
