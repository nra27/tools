SC03 2D Thermal Modelling

Section 4: Thermal Boundary Conditions (3)

  • Thermal Stream 3

  • Thermal Stream 4

Thermal Stream 3

Use the Last Command option to select Thermal - Streams

Complete the Thermal Stream panel as shown below

 

 

  • The factor in the HTC expression takes account of the representation of a 3D feature (i.e. holes on a PCD) on a 2D axisymmetric model. For further information see: Best Practices - HTC Factoring for Internal Lines & Thickness Properties.

  • More information is available about mfcd.

 

Then point and click on points E and F, (Make sure that you click inside the geometry).

 

Thermal Stream 4

Use the Last Command option to select Thermal/Streams

Complete the Thermal Stream panel as shown below

 

Then point and click on points G and H.  (Make sure that you click inside the geometry).

 

 

  • Boundary conditions are applied to a face enclosed by 'boundary lines'. Use the Display - Display Settings command to switch boundary lines on and note the boundary lines inside the cooling holes.

 

Thermal Void

Thermal voids are used when a portion  of the boundary is in contact with a fluid that is deemed to have negligible thermal capacity.

Select Voids... from the Thermal Menu and complete the Thermal Void panel as shown on the right

 

Then point and click on points A and B.

 

 

  • The power input term allows an additional heat input to the void. In this case 5% of duct 1 flow is fed into the void calculation.

  • More information is available about ncy.

 

Property Display Settings

Open the Display Menu by clicking on Display and then choose Property Display from the menu.

Then select the Thermal tab.

Now you can select different colours to highlight the different boundary conditions that you have just applied.

First, select a colour from the palette

Then click on a Col button next to the boundary condition required. (You have just defined Conv. Zone, Duct, Void and Stream conditions).

Then click on OK to apply the different settings.
