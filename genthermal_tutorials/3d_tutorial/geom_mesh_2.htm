Automatic Meshing

The automatic meshing in SC03 generates either:

 

  • 3D 10 noded (quadratic) tetrahedral elements or:

  • 2D 6 noded (quadratic) triangular elements.

To generate a 3D mesh, the major steps completed automatically by SC03 are:

  1. Geometry recognition

  2. Node generation

  3. Surface triangulation

  4. Solid tetrahedralisation

Go to the Thermals homepage to follow the Quick Link to the SC03 Users Guide for further details of the automatic meshing control.

We will now generate a mesh on the solid geometry we read into SC03

Open the Mesh File Menu by selecting 'Mesh' with the left mouse button


Select the 'Generate Mesh' command

The Meshing Control Panel will then be displayed

 

  • Make sure that the default mesh settings are used as shown on the right

  • then select OK with the left mouse button.

The 3D geometry will then be automatically meshed, and as the mesh generation progresses the main meshing steps can be observed.

Every SC03 command panel includes a help button. Click on the help button at any time to see the Users Guide help section for that command.

 

If the mesh is not visible open the display settings panel by clicking on Display and selecting Display Settings from the menu, then select Mesh Outlines

Meshing Control

A number of meshing options are available to control certain aspects of the automatic mesh generation.

Go to the Thermals homepage to follow the Quick Link to the SC03 Users Guide (Section 6.5) for further details of the automatic meshing process.

If the automatic meshing fails or the mesh is inadequate for a specific problem (consult a local expert for guidance on the mesh density requirements) these controls can be used to improve the meshing.

We will remesh our geometry to improve the mesh density.

Select the Generate Mesh command again.

Confirm it is OK to overwrite the existing mesh.

Change the No. of Largest Faces value in the Mesh Control Panel to 10.

Select OK to accept the revised meshing controls.

The 3D geometry will then be remeshed using the revised meshing options. Note how the mesh density has changed relative to the previous mesh.

Go to the Thermals homepage to follow the Quick Link to the SC03 Users Guide (Section 6) for further details of how to work with data panels
