Basic 3D Geometry

Before beginning this section you should already have an SC03 session open.  If not, return to the section on How to start an SC03 session.

Did you know...?

You can create and modify 2D geometry in SC03 but 3D geometry must be imported from another source.

 

In this section we are able to use 3D geometry for the problem described in the User Guide.

The geometry has already been produced for this tutorial and is stored as a .pm file, so the analyst need only read in this .pm file.

The starting point for 3D automatic analysis is a CAD solid model.  This is generally read into SC03 as a STEP file.

Read an SC03 Database

The best way to learn how to interact with SC03 via the mouse and keyboard is to practice using the program itself.

Open the Model Menu by selecting Model with the left mouse button

then Select the Open File command with the left mouse button

Select Input Database

We will now select an existing SC03 database:

  • First Type /usr/rreng/data/sc77/ and press enter,
    If you cannot access this directory, the required files are stored on the web.

  • then select tutorial3d.pm from the file menu or alternatively type in tutorial3d.pm and click on OK

If you have any problems picking up this model, please contact Chris Barnes (49721)

Full details of all menu commands are provided in the SC03 Training Section:

'Learn about SC03 Commands'

  • If you want to learn more about the File Menu commands now, go to the Thermals homepage to follow the Quick Link to the SC03 Users Guide (Section 6.1)

  • If you want to learn more about File Selector Panels, go to the Thermals homepage to follow the Quick Link to the SC03 Users Guide (Section 6.1)

 

Initial Display

The CAD geometry should now be visible in the SC03 display window:

SC03 saves display settings such as colours and display options (e.g Geometry on/off) so that subsequent program sessions are started with previously defined settings and preferences. Your display may therefore look slightly different.

 

If no geometry is currently visible in the display window:

  1. Select the Display Menu

  2. Select the Display Settings command

  3. Select the Options panel tab

  4. Turn On the following Display Settings:
    G - Geometry
    P - Physical Geometry

 

External Meshing

Although the usual route for 3D automatic analysis is to use 3D solid geometry and the SC03 automatic mesh generation capabilities, it is possible to generate a 3D mesh via an external program (e.g. PATRAN) and use this as this starting point.

In this case the mesh is imported to SC03 via the Open File - Mesh command. the mesh must be in PATRAN neutral file format.

The main differences between using CADDS 3D geometry and external 3D meshes are summarised below:

3D Geometry

  • 3D solid geometry from CADDS

  • Solid geometry meshed automatically by SC03

  • Automatic mesh refinement possible to improve accuracy of results

  • Boundary conditions applied to the geometry - little scope for changing where boundary conditions are applied (without modifying geometry in CADDS)

External 3D Mesh

  • 3D mesh from external mesh generation program (e.g. PATRAN) using brick, wedge and tetrahedral elements.

  • Manual meshing required.

  • No automatic mesh refinement possible in SC03 analysis.

  • Boundary conditions applied to the mesh element faces contained within boundary lines - wide scope for changing where boundary conditions are assigned (use Model/ Define Boundaries command).
