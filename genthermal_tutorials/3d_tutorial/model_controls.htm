SC03 3D Thermal Modelling

Section 5: Model Controls

Objectives of this Section

When you have completed this section you will be able to:

  • explain the meaning of the term "domain" in the context of 2D and 3D modelling

  • list the domain properties that need to be set in the Domain Control panel

  • state the functions performed by the Thermal Analysis controls

  • describe what facilities exist in SC03 for the checking of analysis models

Learning Points of this Section  

  1. Defining the type of analysis to be completed 

  2. Defining the analysis accuracy

  3. Defining domain properties in the Domain control panel

  4. Model Input Checking

Global Control

The type of analysis to be completed and the associated accuracy must now be defined.

Open the Model Menu by selecting Model, then select Global Control

 

Complete the Global Control panel to assign a title and ensure that Thermal is selected for the Analysis Type.

Click the Thermal Control button and ensure that the settings are as shown below.

 

Select OK for the Thermal Control and Global Control panels.

 

Thermal Controls

The thermal analysis controls are further detailed below:

Steady State

The 'steady state' option allows a thermal analysis to be speeded up if only a final, steady state solution is of interest. It has the effect of turning off any time step reductions necessary to follow transient behaviour.

Coupled

The 'coupled' option is necessary for a thermo-mechanical analysis which has thermal boundary conditions that reference the structural solution using reference points.

Maximum Temperature

The maximum temperature should be set above the largest solid or fluid temperature expected. An arbitrarily large value should not be used.

Time Stepping Accuracy

The time stepping accuracy controls the number of time steps taken in a transient thermal analysis.

Refinement Accuracy

The refinement accuracy defines the required solution accuracy.The mesh will be refined to achieve the required accuracy.

Domain Properties

The properties associated with each domain must now be defined.

A domain is simply an individual solid (3D) or region (2D) defined by the geometry. For 3D models a domain is often a separate component.

When an external mesh is being used for analysis and no geometry is present, the domain is a collection of elements. Domains may be configured using the Model/Assign Domains command.

 

Open the Model Menu by selecting Model then Select Domain Control
