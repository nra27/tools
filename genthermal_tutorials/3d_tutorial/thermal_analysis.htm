SC03 3D Thermal Modelling

Section 6: Thermal Analysis

Objectives of this Section

When you have completed this section you will be able to:

  • describe the two main methods for completing a thermal analysis

  • state what action should be taken if an analysis has failed

Learning Points of this Section

  1. Submitting models for analysis

  2. Completing an interactive analysis

  3. Analysis failures

 

Complete an Analysis

There are 2 main methods for completing an analysis: Submit and Run

1) Submit

The Analysis/Submit command is used to request an analysis on a batch computer server. This could be:

  • A departmental computer server (e.g. Silicon Graphics Origin 2000)

  • Other LAN connected UNIX machine

2) Run

The Analysis/Run command is used to request an analysis on the current local machine.

Open the Analysis Menu then select Run

Initially an Output Results File panel will appear, prompting for the name of a results file (.fem file) to which to write results.

Enter the name of a results file to be used.

Extensive checking of the analysis model input and controls is then performed, to check that all boundary conditions can be evaluated and all necessary input for the analysis has been provided.

If the input check is successful, a confirmation panel is given, which summarises the type and accuracy of the analyses to be performed.

If this is correct, select OK to start the analysis. CANCEL will stop the run request.

If input errors are detected, it will be necessary to check the analysis model.

A summary of the analysis progress is given in the SC03 output window. The analysis summary from a batch run is held in the SC03 diagnostics (.diag) file.
