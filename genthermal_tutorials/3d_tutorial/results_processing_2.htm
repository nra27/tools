Time Plots

The Get Values capability provides the ability to determine a result value at a specific point on the model at individual times during the analysis cycle.

The Time Plots capability enables a graph of the result at a specific point on the model to be generated for the complete analysis cycle.

To generate a Time Plot:

Open the Results Menu by selecting Results with the left mouse button then select the Time Plots command

The cursor will change to an arrow, which - when clicked over the contoured results - will open the following panel:

If required, the parameters to be plotted may be changed. In this case a Time Plot of temperature (K) vs. time (t) is to be generated.

Select OK on the Time Plotting Expressions panel.

The Time Plot graph will then be generated.

This is a typical Time Plot.   Your plot may look slightly
different, depending on where you selected your points.

The Time Plot graph may be manipulated and displayed in a number of different ways using the Graph Window Menu commands.

Full details of all graph commands are provided in the SC03 Training Section: 'Learn about SC03 Commands'.   If you want to learn more about Graph commands now, go to the Thermals homepage to follow the Quick Link to the SC03 Users Guide (Section 6.9 Graphs)

 

Define Graph Layouts

SC03 also supports a general graphing capability that enables results values to be combined with each other or with external data such as engine measurements. We will produce a simple graph layout to display the temperature difference between the 2 shafts in the tutorial model.

The first step is to define the points at which results are to be graphed. This is done using Reference Points:

Open the Model Menu by selecting Model with the left mouse button then select the Reference Points command.

 

 

In the Reference Points panel:

Enter a point name (e.g. 'outer'), click OK and detect a point on the outer shaft with the left mouse button

Enter another point name (e.g. 'inner'), click OK and detect a point on the inner shaft of the model.

Click CANCEL to end the Reference Points command.

We can now refer to results at these points within user defined graphs.

The next step is to define the graph layout to be used.

Open the Graph Menu by selecting Graph with the left mouse button then select the Define Graph Layouts command.

The Graph Layouts panel should now be displayed:

The Graph Layouts panel contains 3 tabbed panels to select existing or define new Graphs, Sub-Graphs and Plot Variables.

A graph layout consists of a graph which may contain a number of subgraphs, each of which may in turn contain several plot variables.

Complete the Graph Layouts panel as shown below:

Now click the SubGraph Layout tab at the top of the panel.

The Selector button can be used to pick an existing graph layout.
The Delete button can be used to delete an existing layout

 

Complete the SubGraph Layouts panel as shown below:

Then click the Plot Variables tab at the top of the panel.

 

The Select button can be used to pick an existing graph layout.
The Delete button can be used to delete an existing layout

 

Complete the Plot Variables panel as shown below:

  • Note how the plot variable expressions refer to reference points.

The graph layout definition is now complete.

Select OK on the Graph Layouts panel.

Open the Graph Menu by selecting Graphs with the left mouse button then select the Plot Graph Layouts command.

Select Graph G1 then click OK with the left mouse button.

A graph of the temperature difference between the inner and outer shafts will then be generated...

The resulting graph layout is shown below:

Your graph may differ from the one shown above, depending on where you placed your reference points.

The graph may again be manipulated and displayed in a number of different ways using the Graph Window Menu commands.
