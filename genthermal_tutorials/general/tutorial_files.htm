Tutorial Files

The following files may be required to complete the 2D tutorial. The easiest method of retrieval is to access the files from the directory /usr/rreng/data/sc77. 

If you are unable to access these files, right-click on the hyperlink to tutorial_files.zip and use 'save target as'. The file should then be unzipped. You may need to use the file transfer best practice to transfer the files to UNIX.

  • tutorial_files.zip 

  • Absolute/Relative tutorial files (general example exercise 8)

 

2D Thermal Analysis Tutorial

For the SPAN data used in VW34, the following files are required:

  • 8357_HP5_DELY_01_12.SP70_BASE

  • 8357_HP5_DELY_01_12.BASIC.SP70_OUTPUT

or, if you wish to skip the vw34 (cycle definition) section, you will need the following file:

  • tutorial2d.span.bdd

 

3D Thermal Analysis Tutorial

The 3D model file is:

  • tutorial3d.pm

For the SPAN data used in VW34, the following files are required:

  • 8357_HP5_DELY_01_12.SP70_BASE

  • 8357_HP5_DELY_01_12.BASIC.SP70_OUTPUT

or, if you wish to skip the vw34 (cycle definition) section, you will need the following file:

  • tutorial2d.span.bdd

 

General Example Tutorial

The general example model file is:

  • general.example.start.pm

Absolute/Relative temperature tutorial (Exercise 8):

  • Starting model (general.example.ABSREL.zip)

  • Final model with all boundary conditions applied (general.example.ABSREL.complete.zip)

If you are having trouble accessing these files, contact Chris Barnes (49721)
