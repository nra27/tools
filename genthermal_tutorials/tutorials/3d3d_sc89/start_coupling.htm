SC03 Tutorial for 3D-3D Coupling with SC89

10. Starting the coupling

Save the changes you have made to the model.

This model includes a command to keep the iterated CFD models so you can view them afterwards.

If in future you start a another coupling procedure from the beginning, this command will not be present by default.  In order to keep the CFD files you would carry out the following procedure:

Enter the monitor window (SC03 Output) through <Analysis/Enter monitor> and type (pressing return after each line):

create

P89_KEEP_CFD

1 0 0 0

1

e (to exit the monitor window)


The CFD files which are iterated during the coupling for this exercise will be stored in your computer's /usr/tmp/ directory, while the original copies will be left unchanged.  (If you would like the temporary CFD files stored in a different location, quit SC03 and type Setenv TMPDIR [pathname] at the prompt and then restart SC03.)

Click the Go button on the toolbar to run the process like other SC03 problems.

During the coupling process FLUENT will be run many times and the two programs exchange information iteratively until results of the specified accuracy are generated.  During this time you will see FLUENT repeatedly begin, execute and close and the residuals chart will be plotted along with the SC03 temperature monitor.

If you find the appearance of extra windows irritating, change the Run CFD In Separate Window To Monitor Residuals toggle to No in the Misc tab of the CFD Boundary Conditions Definition panel.

For this model the whole coupling process should take around 3 days depending on the speed of your computer.
