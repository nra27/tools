SC03 Tutorial for 3D-3D Coupling with SC89

9. Setting up the boundary conditions (3)

The dummy streams which are associated with the inlets and outlets of the CFD models are applied to the SC03 model in the normal way.  

Select <Thermal/Stream> and fill in the first index as shown below

('There is 1 more BC' refers to the mesh condition also applied to this surface.)

In stream 1 the Inlet Temp field is used to define the inlet temperature for the annular region between the disc and shaft according to the cycle environment parameter t26.  

Apply this stream to the surface indicated in the image and then apply the three further streams described in the table.
