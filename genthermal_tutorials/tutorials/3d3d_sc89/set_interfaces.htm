SC03 Tutorial for 3D-3D Coupling with SC89

6. CFD interfaces

Click on the Interface tab of the CFD Boundary Conditions Definition panel, which is used to define the inlets and outlets for the coupling.

As you can see below, the names of the inlet and outlet boundary conditions of the CFD model have been entered into the SC03 panel (in this case the inlet and outlet boundary conditions were called 'inlet' and 'outlet' but this may not always be the case).

 

Copy these details into the panel.  Enter the same information for the 3doutside domain, except the inlet expression will be ST1 instead of ST3.

By telling SC03 the name of the CFD Inlet ,it knows where the CFD needs an air temperature input, which will vary during the cycle.  When FLUENT receives this info it then updates its inlet boundary condition and re-iterates.  This process continues for each stage of the cycle.

The expression associated with the CFD Inlet Name is a dummy stream.  This stream is attached somewhere on the SC03 model but has no properties which impact on the results of the analysis.  Instead, it serves as a reference point for the CFD model for the cycle's temperature environment parameter.  This point will be illustrated again when the boundary conditions are defined later in the tutorial.

By giving the CFD Outlets a number, another dummy stream can use the information provided by FLUENT to apply it to other parts of the model if necessary.
