Section 4: Saving ALICE performance data in a BDD file

Objectives of this Section

When you have completed this section you will be able to:

  • Create an analysis cycle

 

Learning Points of this Section

  1. Save a BDD file

Click on one of the learning points above to find specific information or scroll down the page to begin the section

 

Saving a BDD file

Click on the File/Save File menu and choose the BDD tab. Save the file as ALICEtutorial.bdd in the Save Output window.

Click OK. The data is now saved.

This BDD can be used with an SC03 model as required using File/Open file/BDD
