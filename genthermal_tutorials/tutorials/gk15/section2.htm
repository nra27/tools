GK15 (SC03 Input Processor) Interactive Tutorial

Section 2: Loading the ALICE performance data

Objectives of this Section

When you have completed this section you will be able to:

  • choose ALICE database

  • choose performance test, version and extract

 

Learning Points of this Section

  1. Open chosen ALICE database

  2. Find correct performance test (experiment)

  3. Choose test version

  4. Choose test extract

Click on one of the learning points above to find specific information or scroll down the page to begin the section

 

Opening the ALICE database

You will now notice that some of the menus have commands added to them. For example, the File menu now has the added command of Read ALICE experiment.

 

Click on Read Alice Experiment. This will give you a window where you will choose which database to read from.

  • For Bristol users, type THERMALB.

  • For Derby users, type THERMAL.

Click OK.

 

 

Find Correct Performance Test (Experiment)

Having chosen the database, a list of the tests in the database is shown. From this list, please choose test (Exp. Id.) 1800002 and click OK.

In the Units and Scaling Status panel, click OK. No units need to be changed in this case.

 

Choose Test Version

Choose version BASIC in the Version Selection panel:

Click OK.

 

Choose Test Extract

In the Extract Selection panel, choose extract 0002000. The data extract will currently only be used to define the parameters required. Click OK.

 

In the Comment Window, you will see whether the performance data is averaged, minimal or aged (AVE, MIN, AGE). You are asked to type in a comment to give a reference to a suitable report describing the definition of the particular data set you are reading in.

In this particular case, both averaged, minimal and aged data exists - for test reasons only, but it is only the averaged data that is read in. 

Put in a reference for the averaged data and click Save.

 

In the Parameter Name Status window, click ADD/DEL in order to get the parameter list.

The Parameter Name Selection window gives a list of all available parameters in the test. You have two options on how to choose the parameters:

  1. Choose all the parameters (easiest) and click OK:

  1. Read a list of pre-defined parameters from a text file. To do this click on LOAD and choose the text file where your list of parameters is stored. In this case the file is called Test.txt

It will give the following choice of parameters:

Click OK on Parameter Name Status window. Please note that at this point NO data have yet been read in. So far the only thing being done is defining the experiment, units and parameters.
