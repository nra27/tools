GK15 (SC03 Input Processor) Interactive Tutorial

Section 1: Introduction to SC03 Software

Objectives of this Section

When you have completed this section you will be able to:

  • start SC03

  • load the GK15 plugin from the SC03 window

 

Learning Points of this Section

  1. Starting SC03 

  2. Using the mouse to operate menus and access commands

  3. Load the GK15 plugin

Click on one of the learning points above to find specific information or scroll down the page to begin the section

 

Starting SC03

The first step is to start the SC03 software.

If SC03 is already running on your machine, follow this link.

There are several different methods for starting SC03, depending on the type of machine being used and the way in which your operating system has been configured. The recommended method for starting SC03 when you are using it with the training system is to:

Type sc03 in a shell window

The previous command assumes you have a path to the SC03 script directory defined in your login files.

Type /usr/rreng/script/sc03 in a shell window

On the Technical PC, use the SC03 icon on the desktop to start SC03.

When SC03 has started, your display should look similar to this:

The main SC03 display window is used to enter commands, control the program display, and enable visualisation of your SC03 model.

The output window is used by SC03 to give messages to the user.

All SC03 windows may be re-sized to suit individual preferences but it is recommended that the output window is always kept visible. This will allow any program warnings or errors to be seen. 

Program errors are also highlighted in the status bar at the bottom of the SC03 display window.

Using the Mouse

Most commands can be accessed from the pull-down menus, which are operated by the left mouse button.

You can also operate menus from the keyboard, by pressing the Alt key with the menu mnemonic (underlined character). When the menu is displayed, press the mnemonic for the desired command (this time without the Alt key).

Pull-down menus can be 'torn off' by clicking on the dotted line (left mouse button), and can then be positioned as required. They will then remain visible until you close them down, either from the window manager or by pressing the Escape key.

The right mouse button brings up a context-sensitive popup menu. Over the graphics window this gives quick access to the last command and the Screen commands. With the Control key held down, it displays the last 10 commands.

Since you want to load performance data, proceed immediately to loading GK15. Do not open any other files beforehand as this might create problems with SC03.

Loading GK15

You will now use the left mouse button and the file pull-down menu.

Click on the File pull-down menu and go down to Extensions.

 

You will get the following window:

In the Extensions window, you can choose production, trials or development versions of the plugins. You will use the production version, so press Load to get a list of choices.

Click on libgk15.so and thereafter OK. You will then come back to the Extensions window, which now will give you information on the plugin you have chosen. Click the load on start toggle if you wish GK15 to be loaded each time SC03 is invoked. Click OK to close the panel.

You have now loaded the GK15 plugin.
