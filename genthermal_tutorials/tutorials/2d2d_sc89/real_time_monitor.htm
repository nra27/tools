SC03 Tutorial for 2D-2D Coupling with SC89

9. Setting up a temperature monitor

To monitor the coupling process as it runs, a real time temperature plot can be generated.

Click through <Analysis/Real Time Display>, type tmfd3 in the first box and toggle on the Display results during analysis setting.  This chart will show the progress of the coupling by progressively plotting the temperature of reference point fd3 (a point three quarters the radius of the disc).

 

The SC03 and CFD models are now set up and the coupling is ready to run.
