SC03 Tutorial for 2D-2D Coupling with SC89

This tutorial outlines the method for coupling a simple free rotating disc to a surrounding region of air. Through this example you will learn how to use the SC89 plugin and set-up simple FLUENT and SC03 models to be coupled.

The three files required for this tutorial are contained in sc89_2d2d.zip (approx 700KB). You have two options for extracting the files:

  1. Left click on the link and select Open this file from its current location. You can then extract the files to the directory of your choice.

  2. Right click the link and select Save target as to place the file on a LAN drive (Y, Q etc) then use FTP to place it on the H drive before extracting files.

Please note that the SC89 plugin is currently only compiled for SC03 running on a LINUX operating system and only works with FLUENT 6.1.

For further background information about how SC89 works, see the User's Guide, Supplementary Material and SC89 methods document.
